# CWS BBS trend analyses


[![DOI](https://zenodo.org/badge/1121176400.svg)](https://doi.org/10.5281/zenodo.20561464)

The scripts here represent the complete code archive necessary to reproduce the Canadian Wildlife Services annual analyses of population status and trends from the North American Breeding Bird Survey.

The scripts are numbered based on the order of operations.

## Note: 
The model fitting scripts are written to require significant computational resources, running each model across 4 cores in parallel and simultaneously running many species models in parallel. The model fitting process requires 1000s of core-hours. A complete re-run is likely not worthwhile. However, reproducing the analysis for a single species, or a subset of species, or a customized region or model structure is manageable even with a relatively average personal computer. In addition, interested users are welcome to contact the author of the analyses to get access to the stored model output for the most recent annual analysis, if they wish to conduct customized summaries of the model output.
